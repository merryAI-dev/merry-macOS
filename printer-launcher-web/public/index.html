PDF 선택 후 바로 인쇄

PDF를 선택하거나 이 영역에 드래그하세요. Tailwind 기반 로컬 인쇄 화면입니다.

여기에 PDF 드래그
놓으면 첫 페이지를 미리보고 바로 인쇄할 수 있습니다.

첫 페이지 미리보기

브라우저 내장 PDF 렌더링으로 첫 화면을 확인합니다.

맞춤 인쇄

양면, 흑백, 2-up을 원하는 조합으로 고른 뒤 위자드에서 한 번 더 확인합니다.

사용 흐름

1. PDF를 엽니다.
2. `맞춤 인쇄`에서 양면/흑백/2-up을 조합합니다.
3. 위자드에서 설정을 확인한 뒤 마지막 버튼으로만 인쇄합니다.
4. 필요하면 큐 상태를 확인합니다.
